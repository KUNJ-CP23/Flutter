# Flutter - My First Project

Welcome to my **Flutter Lab Programs Repository**! 🚀  
This project contains a collection of **basic Flutter programs** designed for **learning and practice** during lab sessions.  

---

## 📚 Programs Included  

- **TextField and Button Interaction** - Display input dynamically.  
- **Custom Widgets** - Styling widgets with colors, borders, and shadows.  
- **Image Display** - Load images from assets.  
- **Forms and Validation** - Input handling with validation.  
- **Calculator App** - Basic arithmetic operations using buttons.  
- **Layouts** - Row, Column, and Grid layouts with responsive designs.  

---

## 🛠️ Tech Stack  

- **Flutter** - UI toolkit for building natively compiled mobile, web, and desktop applications.  
- **Dart** - Programming language optimized for fast apps.  

---

## 🚀 Getting Started  

Follow these steps to set up and run the project locally:  

### **1. Clone this repository**  
```bash
git clone https://github.com/KUNJ-cp23/Flutter
```

### **2. Navigate to the project directory**
```bash
cd your-Flutter
```

### **3. Install dependencies**
```bash
flutter pub get
```

### **4.Run the app**
```bash
flutter run
```
